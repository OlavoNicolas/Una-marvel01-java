# Una-marvel01-java
# Projeto Java - Exercícios de Lógica e Condições

Este repositório reúne dois pequenos programas em Java que exploram conceitos básicos de programação, como entrada de dados, uso de condições e tomada de decisão. A ideia aqui é praticar lógica de forma simples e direta, simulando situações do dia a dia.

---

## Sobre os Programas

### Alistamento.java

Esse programa simula uma avaliação inicial para alistamento com base no peso de uma pessoa.

Ao ser executado, ele:

* Pede o nome do recruta
* Solicita o peso
* Retorna uma mensagem informando se a pessoa está:

  * Abaixo do peso
  * Dentro do peso ideal
  * Acima do peso

É um exemplo simples de como usar entrada de dados com o usuário e aplicar regras usando estruturas condicionais.

---

### ProjetoRenascimento.java

Esse segundo programa faz uma verificação básica de prontidão para desafios.

Ele considera dois fatores:

* Se a pessoa tem coragem
* Seu nível de saúde física

Com base nisso, o programa decide se a pessoa está pronta ou se ainda precisa se preparar mais.

---

## Objetivo

Esse projeto foi feito com foco em aprendizado. Ele ajuda a entender melhor:

* Como funcionam estruturas condicionais (`if`, `else`)
* Como trabalhar com diferentes tipos de variáveis
* Como criar pequenas regras dentro de um programa

---

## Como Executar

Para rodar os programas, você precisa ter o Java instalado na sua máquina.

Depois disso:

1. Compile os arquivos:
   javac Alistamento.java
   javac ProjetoRenascimento.java

2. Execute:
   java Alistamento
   java ProjetoRenascimento
